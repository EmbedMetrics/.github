# EmbedMetrics

**Developer-first embedded analytics**

EmbedMetrics helps SaaS and product teams ship **native embedded dashboards** with **governed metrics** and **safe versioning**.

Built for teams that need:
- **SDK-native embedding, not iframes** — dashboards render as first-class UI inside the host app
- **Host-app control APIs** — events, callbacks, and programmatic control for real product workflows
- **Governed metrics through a semantic layer** — consistent definitions and access policies across dashboards and questions
- **Safe versioning and rollback** — for dashboards, semantic models, and custom components
- **Extensible components** — built-in components plus support for custom, domain-specific visualizations

**Ask** is available as a complementary interface for natural-language questions, backed by the same governed semantic layer.
